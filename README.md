# 🧠 Digital Minds Coding Academy

A modern, responsive website for **Digital Minds Coding Academy** — empowering students with practical coding skills through structured and interactive learning.

🌐 **Live Site**: [https://aniwen-blessing.github.io/digital-minds-academy](https://aniwen-blessing.github.io/digital-minds-academy)

## 🌟 Features

- ✅ **Responsive design** – works on mobile, tablet & desktop  
- 🌙 **Dark mode toggle** – respects user preference  
- 📩 **Registration form** – powered by [Formspree](https://formspree.io/)  
- 🎯 **Smooth scroll animations** – enhances user experience  
- 🧑‍🏫 **Mentorship & course sections** – highlights your programs  

## 🛠️ Built With

- HTML5
- CSS3 (with custom properties for theming)
- Vanilla JavaScript
- GitHub Pages
